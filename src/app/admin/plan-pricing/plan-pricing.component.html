info Como Funciona o Sistema de Preços

Preço Base

O preço base é o valor mensal do plano. Ao alterar este valor, todos os SKUs são recalculados automaticamente.

Descontos Automáticos

O sistema aplica automaticamente descontos: Semestral (-16%), Anual (-25%).

SKUs Personalizados

Você pode personalizar preços individuais de cada SKU, sobrescrevendo os cálculos automáticos.
